VIM 中使用jumps命令的技巧

很多人都不太清楚VIM内置的jumps命令应该怎么用？

其实很简单，使用jumps命令查看跳转位置堆栈表，然后使用<c-o> 和 <c-i>配合<num>跳转到指定位置。

如：

‘>’表示当前光标位置，只要你看到想要去的地方。
使用<num> + <c-o>，如2<C-o>就跳到~/VimNote/vim/vim-grep.md。
使用<num> + <c-i>，如1<C-i>就跳到uint32 len。
